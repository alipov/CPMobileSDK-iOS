CPMobileSDKOptions

public class CPMobileSDKOptions : NSObject

A CPMobileSDKOptions object defines the behavior and policies to use when starting the mobile SDK.

  • allowAppUsageWithoutCapsuleIntegration

    Displays a button allowing users to disable the mobile SDK and use the application without security or integration with Capsule Workspace.

    Defaults to false.

    Declaration

    Swift

    public var allowAppUsageWithoutCapsuleIntegration: Bool
  • enforceMobilePolicyOnSystem

    Controls whether mobile policy is enforced on system actions, such as pasteboard copy-paste, document interactions, etc.

    When set to false, system actions, such as copy-paste will be performed normally, without policy enforment. It is up to the developer to ensure no data leaks.

    Defaults to true.

    Declaration

    Swift

    public var enforceMobilePolicyOnSystem: Bool
